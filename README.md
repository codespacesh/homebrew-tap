# Homebrew Tap for Codespace

Desktop app for managing [Coder](https://coder.com) development environments.

## Install

```bash
brew install codespacesh/codespace/codespace
```

This automatically taps `codespacesh/homebrew-codespace` and installs the `codespace` formula.

## Usage

```bash
codespace
```
